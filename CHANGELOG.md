# prompt-working-logs Changelog

## [Unreleased]
### Added
- Initial scaffold created from [IntelliJ Platform Plugin Template](https://github.com/JetBrains/intellij-platform-plugin-template)
- Initial public release. Run the Claude Code CLI (installed at `$HOME/.local/bin/claude`) on markdown prompt files from a watched directory, stream the formatted output into a tool window, manage prompt files from a tree view, and toggle response language between English and Japanese. **Supported OS in this release: macOS only.** **Runs the CLI with `--dangerously-skip-permissions`; review your Claude Code `permissions` settings before installing.**
